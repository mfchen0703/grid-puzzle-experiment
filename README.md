## Project Layout

- `experiment1/`: 现有网页实验（原来的前端、API、Vite 配置和依赖）
- `experiment2/`: 预留给下一个网页实验
- `data/`: 被试导出的 CSV 数据
- `model code/`: 拟合与分析脚本、notebook、图表输出

## Experiment 1

进入 `experiment1/` 后运行：

1. `npm install`
2. `npm run dev`

或直接在根目录运行：

1. `npm --prefix experiment1 install`
2. `npm --prefix experiment1 run dev`
